# P4 バックテスト結果 (replay/run.mjs)

対象: data/fixtures/toda_races.json 全2304レース。時系列先頭70%をtrain(1612レース)、残り30%をtest(692レース)として分割（scripts/lib/common.mjs の splitTrainTest、anatomy.mjsと同一分割）。

係数・重み・買い目型は **train専用データで scripts/train_search.mjs により決定**し、testはこのバックテスト一回のみに使用した（testを見て再選択・再調整は行っていない）。

## 採用ロジックの確定内容

- 使用変数（生死判定=生存のもののみ。docs/anatomy.md参照）: 艇番ベースコース事前分布、当地勝率、全国勝率、モーター2連率、ボート2連率、展示タイム順位、級別
- 重み（train座標降下グリッドサーチ、0/0.5/1/1.5/2の総当たりを1変数ずつ2周）:

| 変数 | 重み |
|---|---|
| course_prior（艇番ベース1着率） | 2 |
| 当地勝率 (loc_win) | 1 |
| 全国勝率 (nat_win) | 1.5 |
| モーター2連率 (motor_2r) | 1 |
| ボート2連率 (boat_2r) | 0（グリッドサーチの結果、寄与なしが最良） |
| 展示タイム順位 (exhibition) | 1 |
| 級別 (class) | 2 |

- 買い目型: 上位3艇BOX(6点) + 1位固定で4位を絡める2点（[1,2,4],[1,4,2]）＝計8点。train上で3型（1着固定流し／上位3艇BOX+α／1-2位box流し）を比較し最良だった型を採用（比較表は下記）。

## 正直な評価

testとtrainの差は的中率1.29pt、回収率3.74ptに収まっており、過学習の明確な兆候は見られない。

testの数字を正とする。目標（真自在律35.4%/78.5%、RONDE18.5%）に対する到達状況は上表の通りであり、届いていない場合もそのまま報告する。

特払い・不成立レースは投資と払戻が相殺されるニュートラル扱いとし、上記の的中率・回収率の分母（レース数）から除外した（件数は表の「特払い等除外」列に記載）。

# P4.5 選別層（自信度ティア）バックテスト

confidence(boats) = (1位スコア-2位スコア) + (上位3艇平均スコア-下位3艇平均スコア)。1位の抜け具合と上位/下位集団の分離度を合成した「買いやすさ」指標。

ティア境界（**trainのみ**で算出した confidence の三分位点、scripts/compute_tier_thresholds.mjs）: q33=9.560521526721155, q67=12.618367981334869。C: confidence<=q33 / B: q33<confidence<=q67 / A: confidence>q67。

test計測は1回のみ実施し、この結果を受けた閾値・confidence式の再調整は行っていない。
